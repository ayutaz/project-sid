## 6.3 専門化実験の手法

### 設計

- **エージェント数**: 30体
- **条件統一**: 全エージェントに同一のミッション、性格特性、村落インフラ位置の記憶を付与
- **変数**: 3種類の村落構成（通常、武闘、芸術）— 初期コミュニティ目標のみ異なる

### 役割生成手法

エージェントは相互作用を通じて再帰的に「社会的目標（social goals）」を生成。シミュレーション終了後、**GPT-4o**を用いてエージェントごとの5目標ローリングセットから役割を推論。

### アクション分析

役割内・役割間でアクションカウントを正規化し、頻度分布を視覚化。システム全体で劇的に異なるレートで発生するアクションを考慮。

## 6.4 集団ルール実験の手法

### システム構成

- **構成員投票者**: 25体
- **インフルエンサー**: 3体（賛税 or 反税）
- **遠隔選挙管理者**: 1体
- **警察**: 意図的に除外（システム複雑性の低減のため）

### タイムライン（合計1200秒）

| フェーズ | 時間 | 内容 |
|---|---|---|
| 修正前フェーズ | 0-600秒 | 120秒間隔で5回の課税シーズン（各20秒の課税ウィンドウ） |
| 民主的プロセス | 300-360秒 | 構成員とインフルエンサーがフィードバック提出 |
| 投票 | 420秒 | 構成員が修正案に投票 |
| 修正後フェーズ | 600-1200秒 | 憲法更新後、5回の追加課税シーズン |

### 実験条件

| 条件 | 内容 |
|---|---|
| 完全PIANO + 修正可能憲法 | フル機能での実験 |
| 凍結憲法コントロール | 憲法変更不可の対照条件 |
| ベースラインアブレーション | 最小構成での対照条件 |

各構成に賛税・反税インフルエンサーのバリエーションを設け、設定ごとに**4回繰り返し**実行。

## 6.5 文化伝達実験の手法

### シミュレーションパラメータ

| パラメータ | 値 |
|---|---|
| エージェント数 | 500体 |
| 実行時間 | 9000秒 |
| 環境サイズ | 1000×1200ブロック |
| 町の数 | 6（各33体） |
| 農村エージェント | 302体 |

### 町の配置

Sunny Glade, Woodhaven, Clearwater, Meadowbrook, Hilltop, Riverbend

### エージェント構成

- 手続き的に生成された名前と性格特性
- **パスタファリアン司祭**: Meadowbrookの20体に宗教伝播用の特別な性格条件付け
- 初期インベントリはランダム化

### ミーム分析手法

LLM呼び出しにより2時間ごとにエージェントの統合目標を要約。キーワードベースの文化的ミーム（"eco", "dance", "meditation"等）を抽出し、各エージェントの目標履歴でミーム出現を分析。

### 宗教伝播の追跡

| 分類 | キーワード |
|---|---|
| 直接改宗 | "Pastafarian", "Spaghetti Monster" |
| 間接改宗 | "Pasta", "Spaghetti" |

改宗階層と時空間的伝播を追跡。

## 6.6 評価手法の特徴と課題

### 使用された評価手法

| 手法 | 適用先 | ツール |
|---|---|---|
| アイテム数カウント | 単一エージェント性能 | 直接計測 |
| ピアソン相関分析 | 社会的認知精度 | 統計分析 |
| LLM役割推論 | 専門化 | GPT-4o |
| キーワード追跡 | 宗教伝播 | テキスト分析 |
| LLMミーム抽出 | 文化ミーム | LLM分析 |
| 行動頻度分析 | 税遵守率 | 直接計測 |

### 方法論的課題

- **LLM依存の循環性**: 役割推論・ミーム分析にLLMを使用しており、LLMベースのエージェントの行動をLLMで評価するという循環的構造
- **人間評価との比較なし**: LLM推論結果の人間評価による検証が不足
- **統計的検出力**: 各条件4回の繰り返しは限定的
- **キーワード追跡の単純性**: 宗教伝播の深さ・理解度を測定していない
