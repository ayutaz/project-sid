# 5. 文明的進化プロセス

[← 前: マルチエージェントの進化](./04-multi-agent.md) | [トップ](./index.md) | [次: 研究手法の詳細 →](./06-methods.md)

---

## 概要

Section 5は、AIエージェントが原始的な集団行動を超えて、専門化された役割、集団的統治、文化伝達を特徴とする複雑な文明へと進歩する能力を評価している。人間の歴史的文明発展と対応する複数の次元にわたって実験を行った。

---

## 5.1 専門化（Specialization）

### 実験設計

| パラメータ | 設定 |
|---|---|
| エージェント数 | 30体（同一の性格と共同体目標） |
| 時間 | 約20分間 |
| 役割推定方法 | 5〜10秒ごとのソーシャルゴールをGPT-4oで分析 |
| 村落構成 | 3種類（通常、武闘、芸術） |

**有効な専門化の基準**:
- 自律的な役割選択と遷移（Autonomy）
- 明示的制約なしに相互作用から創発すること（Emergence）
- 選択した専門に一致する行動パターン（Behavioral alignment）

### 主要な発見

1. **自発的役割分化**: 明示的指示なしに6種類以上の役割に自己組織化

| 創発した役割 | 対応する行動パターン |
|---|---|
| 農民（Farmer） | 種子収集 |
| 鉱夫（Miner） | 採掘活動 |
| 技術者（Engineer） | 建築・修理 |
| 警備員（Guard） | フェンス制作 |
| 探検家（Explorer） | 探索活動 |
| 鍛冶屋（Blacksmith） | 金属加工 |

2. **役割の持続性**: 個々のエージェントの役割が時間経過とともに安定
3. **文脈依存性**: 軍事的社会→「偵察兵」「戦略家」、芸術的社会→「学芸員」「収集家」

### アブレーション結果

- 社会認識モジュール除去で専門化が**失敗**（役割が均質かつ非持続的に）
- 役割エントロピー（role entropy）: 社会モジュールありの場合に増大し、多様性を確認

## 5.2 集団ルール（Collective Rules）

### 実験設計

| パラメータ | 設定 |
|---|---|
| 総エージェント数 | 29体 |
| 構成員投票者 | 25体 |
| インフルエンサー | 3体 |
| 選挙管理者 | 1体 |
| 初期法規 | 所持品の20%を税として納付 |
| 実験時間 | 1200秒（20分） |

**注意**: 警察による執行は意図的に除外（システム複雑性の低減のため）

### 民主的プロセス

```
構成員がフィードバック → 選挙管理者が修正案処理 → 民主的投票 → 憲法更新
```

**タイムライン**:

| フェーズ | 時間 | 内容 |
|---|---|---|
| 修正前 | 0-600秒 | 120秒間隔で5回の課税シーズン |
| 民主的プロセス | 300-360秒 | フィードバック提出、修正案処理 |
| 投票 | 420秒 | 構成員が修正案に投票 |
| 修正後 | 600-1200秒 | 憲法更新後、5回の追加課税シーズン |

### 主要な発見

| 条件 | 結果 |
|---|---|
| 賛税影響力者あり | 税率維持〜引き上げ |
| **反税影響力者あり** | **税率 20% → 9% に低下** |
| 凍結憲法コントロール | 行動変化なし |
| ベースラインアーキテクチャ | 双方向的変化が阻止 |

- 構成員は初期税法（20%）を遵守
- 憲法修正が税の遵守行動に**直接反映**
- 影響力者の態度が投票行動を形成（感情伝播）

## 5.3 文化ミーム（Cultural Memes）

### 実験設計

| パラメータ | 設定 |
|---|---|
| エージェント数 | 500体 |
| 配置 | 6つの町に各33体（計200体）、農村地域に300体 |
| 環境 | 1000×1200ブロックの領域 |
| 実験時間 | 約9000秒（2.5時間） |
| 町の名前 | Sunny Glade, Woodhaven, Clearwater, Meadowbrook, Hilltop, Riverbend |

### 主要な発見

1. **人口密度の閾値効果**: 農村部は町よりも有意に少ないミームを生成。社会的接続性の一定レベルがミーム伝播に不可欠

2. **地域的特異性**: 異なる町が異なるミームを発展

| 町 | 発展したミーム文化 |
|---|---|
| Woodhaven | エコテーマ |
| Clearwater | いたずら（pranking）文化 |

3. **時間的変動**: 個々の町内でミームの人気度が時間とともに変動
4. **共存性**: 単一社会内で複数のミームが同時共存

## 5.4 宗教（Religion）— パスタファリアニズムの伝播

### 実験設計

- **設定**: Meadowbrook村に20体のパスタファリアン（Pastafarian）司祭を配置
- **追跡方法**: エージェント発話における宗教的キーワードの出現を追跡

### 改宗レベルの分類

| レベル | 定義 | キーワード |
|---|---|---|
| 直接改宗 | 宗教的キーワードを明示的に使用 | "Pastafarian", "Spaghetti Monster" |
| 間接改宗 | 関連用語を組み込む | "Pasta", "Spaghetti" |

### 主要な発見

| 発見 | 詳細 |
|---|---|
| **持続的拡大** | 直接・間接の改宗は2時間にわたり飽和なしに増加 |
| **地理的伝播** | 司祭と改宗者の町間移動に伴い空間的に拡大 |
| **影響圏の拡大** | 改宗者で囲まれた非重複面積が時間とともに増加 |
| **段階的変換** | 初回曝露（critical exposure）→ 強化（reinforcement）の段階パターン |

宗教伝播は**疫学モデル（SIRモデル）**の感染拡大ダイナミクスと構造的に類似。司祭が「感染源」、改宗者がさらなる伝播の「媒介者」として機能。

## 5.5 人類文明との比較

### 類似点

| 次元 | 人間文明 | AI文明（Project Sid） |
|---|---|---|
| 分業 | 経済的効率性から自然発生 | 社会的相互作用から自然発生 |
| 法制度 | 民主的プロセスによる制定・改正 | 投票による憲法修正と行動変化 |
| 文化伝播 | 都市部で加速、地域性あり | 人口密度依存、町ごとの特異性 |
| 宗教的伝播 | 宣教活動による地理的拡大 | 司祭エージェントによる空間的伝播 |
| 影響力者の役割 | 世論形成への影響 | 投票行動と法改正への影響 |

### 相違点

| 次元 | 人間文明 | AI文明 |
|---|---|---|
| 時間スケール | 数百〜数千年の蓄積 | 数分〜数時間で創発 |
| 世代間継承 | 遺伝的・文化的な世代間伝達 | 単一世代内の学習のみ |
| 動機の深さ | 生存・繁殖・欲求など多層的 | プロンプトベースの動機付け |
| 制度的記憶 | 先例・慣習・文書による長期蓄積 | シミュレーション期間内の短期記憶 |
| 技術発展 | 累積的知識による技術革新 | 環境内の既存機能の利用 |
| 紛争と対立 | 資源競合による戦争・革命 | 限定的な社会的摩擦のみ |
| 規模の複雑性 | 数十億人規模の多層的制度 | 最大1000体程度、単層的制度 |

## 5.6 文明的ベンチマークの設計思想

### 提案されたベンチマーク次元

| 次元 | 測定指標 |
|---|---|
| **専門化** | 役割エントロピー、役割持続時間、行動-役割相関 |
| **制度的統治** | 税遵守率、憲法修正の行動への反映度 |
| **文化的伝播** | ミーム発生数、地理的伝播範囲、改宗率 |
| **インフラ発展** | 制度の機能的運用度 |

### 設計哲学

- **人間文明との対応**: 狭いタスク指標ではなく、人間文明の主要次元を反映
- **創発性の重視**: 事前プログラムではなく自発的に創発する現象を測定
- **スケーラビリティ**: 数十体から数千体への拡張を前提
- **多次元評価**: 単一指標ではなく複数次元の同時評価

## 5.7 定量的データまとめ

| 実験 | エージェント数 | 時間 | 主要定量結果 |
|---|---|---|---|
| 専門化 | 30体 | 20分 | 6種類以上の役割が自発的に創発 |
| 集団ルール | 29体 | 20分 | 税率20%→9%（反税条件） |
| 文化ミーム | 500体 | 2.5時間 | 町ごとに固有のミームパターン |
| 宗教伝播 | 500体（司祭20体） | 2時間超 | 飽和なしに継続的改宗増加 |
