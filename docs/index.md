# Project Sid: Many-agent simulations toward AI civilization
## 論文詳細分析ドキュメント

> **論文**: Project Sid: Many-agent simulations toward AI civilization
> **著者**: Altera.AL (2024)
> **arXiv**: [2411.00114](https://arxiv.org/abs/2411.00114)
> **分析日**: 2026-02-23
> **分析手法**: 5名の専門家エージェントによる各セクション独立調査・統合

---

## 概要

Project Sidは、10〜1000体以上のAIエージェントがMinecraft環境内でどのように振る舞い、文明的進歩を遂げるかを実証した研究である。**PIANO（Parallel Information Aggregation via Neural Orchestration）**アーキテクチャを提案し、エージェントがリアルタイムで人間や他のエージェントと対話しながら、複数の出力ストリーム間で一貫性を維持することを可能にした。

### 主要な成果

| 成果 | 詳細 |
|---|---|
| PIANOアーキテクチャ | 並行性+一貫性を両立する神経科学着想の認知アーキテクチャ |
| 単一エージェント | 30分で平均17種のアイテム獲得、ダイヤモンド30分で達成 |
| マルチエージェント | 50体社会で社会的認知精度 r=0.807、集合知の発現を確認 |
| 文明的進化 | 6職種の自発的分化、民主的統治、文化ミーム伝播、宗教伝播を実証 |

## 全実験サマリー

| 実験 | エージェント数 | 時間 | 主要定量結果 |
|---|---|---|---|
| 単一エージェント | 25体 | 30分 | 平均17種のユニークアイテム |
| 長期アイテム収集 | 49体 | 4時間 | 約320種で飽和 |
| 感情追跡 | 1+3体 | - | 感情変動の正確な追跡 |
| 食料配分 | 1+4体 | - | 感情推測に基づく選択的配分 |
| 社会認知 | 50体 | 4時間超 | r = 0.807（認知精度） |
| 専門化 | 30体 | 20分 | 6種類以上の役割創発 |
| 集団ルール | 29体 | 20分 | 税率20%→9%（反税条件） |
| 文化伝達 | 500体 | 2.5時間 | 町ごとの固有ミームパターン |
| 宗教伝播 | 500体 | 2時間超 | 飽和なしの継続的改宗 |
